Quiz App

This quiz app was built with a New Orleans, LA theme and has five questions. After each question is answered immediate feedback is given. If the user answers correctly they get a thumbs up. If the wrong answer is chosen, the user is given the correct answer. At the end of the quiz, a final overall score is given with a chance to start over and take the quiz again.

Includes title, image and description, as well as a start button.
The correct answer is shown if the user answers wrongly. Also, user can see their progress.
A final score is given when the quiz is finished and a chance to start over.
